Portale multifunzione

Questo è un portale di prova dove verranno inserite le varie modalità di login (google, facebook, instagram, ecc..)

Il BE di questa applicazione è impostata con una serie di microservizi sviluppati con spring boot

Verrà utillizzato spring security per tutta la parte di autenticazione e autorizzazione

Verranno creati componenti di compilazione pratica per utilizzare modalità di compilazione con trasmissione un JSON
